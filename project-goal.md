# MountGuard for Mac 产品规格说明书
## 一、产品概述
MountGuard for Mac 是一款专为 macOS 打造的**开源外接磁盘管理工具**，核心聚焦移动硬盘/U盘/外接存储设备的「自动安全挂载、全格式无损读写、无占用安全移除」三大核心场景，彻底解决现有 NTFS 挂载工具不稳定、数据易损坏、安全移除难、权限要求高、闭源不透明的行业痛点。

产品全程基于 macOS 原生磁盘框架开发，不修改系统文件、无需关闭 SIP、无内核扩展风险，提供极简流畅的原生 GUI 交互，原生兼容 Apple Silicon 全系列与 Intel Mac，可打包为 DMG 安装包分发，完全开源托管于 GitHub，兼顾新手零门槛使用与专业用户自定义需求，核心目标是成为 Mac 平台最安全、最稳定、最好用的外接磁盘管理工具。

## 二、核心定位与差异化优势
### 核心定位
Mac 外接磁盘全生命周期安全管理工具，以「**磁盘与数据安全**」为第一原则，兼顾极致易用性与高阶扩展性，彻底解决跨平台磁盘读写、安全移除的核心痛点，做闭源商业软件的开源平替，同时实现对 Mounty 等免费工具的体验碾压。

## 三、目标用户与系统兼容要求
### 目标用户
1. **核心用户**：跨Windows/Mac办公的职场人、设计师、剪辑师、摄影师，频繁使用外接磁盘传输大文件，对读写稳定性、数据安全性、便捷移除有强需求；
2. **泛用户**：Mac新手，不懂终端挂载命令，担心操作失误损坏磁盘/系统，需要零门槛可视化工具；
3. **开源社区用户**：追求透明、可定制、无捆绑的工具，有二次开发需求；
4. **企业/团队用户**：需要统一、安全、可管控的外接磁盘管理工具，用于内部办公设备标准化部署。

### 系统兼容要求
- 最低系统版本：macOS 12.0 Monterey 及以上（覆盖95%以上现役Mac设备）
- 架构支持：Apple Silicon（arm64）全系列原生支持，Intel x86_64 原生支持，无需Rosetta 2转译
- 硬件支持：全系列MacBook、iMac、Mac Studio、Mac Pro、Mac mini
- 权限要求：仅需「磁盘管理权限」+ 可选「辅助功能权限（用于进程占用释放）」，无需root权限，无需修改系统配置，无需关闭SIP

## 四、核心功能模块详解
### 模块一：磁盘自动识别与智能挂载模块
核心目标：无感、稳定、容错性强的磁盘挂载能力，避免手动操作，杜绝挂载失败导致的读写异常。
1. **实时热插拔监听**
    - 基于 macOS 原生 DiskArbitration 框架，实现外接磁盘插入/拔出事件的实时监听，响应延迟＜100ms，全事件驱动无轮询，不占用系统资源。
    - 自动屏蔽Mac内置系统磁盘、恢复分区、VM分区，默认仅展示外接可移动磁盘，从根源避免用户误操作系统磁盘导致崩溃。
2. **全维度磁盘信息识别**
    - 自动识别并可视化展示磁盘核心信息：设备名、卷标、文件系统格式、总容量/已用/可用空间、挂载状态、读写权限、硬件接口（USB-C/USB-A/Thunderbolt）、磁盘UUID、SMART健康状态、是否为可引导磁盘。
    - 基于UUID唯一识别磁盘，不受卷标修改、盘符变化影响，确保规则绑定的永久性。
3. **智能自动挂载规则引擎**
    - **默认规则**：插入外接磁盘后，自动识别文件系统格式，自动以「可读写模式」安全挂载，挂载成功后推送系统通知，点击通知可直接打开磁盘根目录，全程零操作。
    - **自定义规则**：支持针对单个磁盘（UUID绑定）设置专属规则：开机自动挂载、插入自动挂载、只读模式挂载、挂载到指定目录、挂载后自动执行预设脚本（如备份）、是否允许Spotlight索引/Time Machine备份（避免系统进程自动占用）。
    - **黑名单规则**：支持将特定磁盘加入黑名单，插入后不自动挂载、不提示，避免恶意磁盘/无关磁盘自动加载。
4. **挂载异常容错与修复**
    - 挂载失败时，自动识别失败原因（如未正常推出、文件系统损坏、权限不足、进程占用），输出可视化中文提示，而非晦涩的终端错误代码。
    - 针对「未正常推出」的磁盘，自动执行macOS原生文件系统一致性检查（fsck），检查通过后自动重试挂载，无需用户手动执行终端命令。
    - 异常场景兜底：文件系统检查不通过、磁盘存在坏道/SMART异常时，自动强制以只读模式挂载，禁止写入，避免进一步损坏磁盘与数据，同时推送告警通知。
    - 全流程日志记录，挂载失败可一键导出日志，方便问题排查。

### 模块二：全格式安全读写模块
核心目标：无损、稳定、高性能的磁盘读写能力，绝对不损害磁盘、不丢失数据，这是产品的核心底线。
1. **全文件系统格式支持**
    - **原生完美支持**：NTFS（macOS原生驱动，无需第三方组件）、FAT32、exFAT、APFS、HFS+、HFS，全程基于系统原生驱动，无兼容性风险。
    - **扩展可选支持**：ext2/ext3/ext4、Btrfs、XFS，兼容开源macFUSE框架，用户可一键开启支持，无需手动安装配置，无强制捆绑。
2. **NTFS安全读写核心优化（核心竞争力）**
    - 前置校验机制：NTFS磁盘挂载前，强制执行原生fsck_ntfs文件系统一致性检查，仅当检查通过后，才执行可读写挂载，彻底杜绝文件系统存在错误时写入导致的数据损坏（这是Mounty等工具的核心缺陷）。
    - 安全挂载参数：基于macOS原生mount_ntfs框架，优化挂载参数，默认采用「安全写入模式」，开启同步写入机制，确保数据实时落盘，避免断电/意外拔盘导致的文件损坏；高级用户可手动开启「高性能模式」，适配大文件高速传输场景。
    - 权限自动适配：自动处理挂载后的文件权限，默认当前用户拥有完整读写权限，无需手动执行chmod命令，解决「挂载后只能读不能写」的常见问题。
    - Windows兼容保障：完全兼容Windows NTFS权限体系，不修改文件安全属性，避免磁盘在Mac写入后，拿到Windows上出现权限异常、文件无法打开的问题。
3. **读写全流程安全保障**
    - 写入完整性校验：支持文件写入后自动MD5/SHA256校验，确保文件写入完整，无半截文件、损坏文件问题，大文件可选择后台校验，不影响前台传输速度。
    - 写入缓存实时监控：实时监控磁盘写入缓存状态，缓存未完全落盘时，禁用安全移除按钮，实时展示缓存写入进度，从根源杜绝缓存未落盘拔盘导致的数据丢失。
    - 防误操作保护：默认禁用格式化功能，需用户手动在设置中开启并二次确认，避免新手误操作格式化磁盘；所有高危写入操作均有明确的风险提示与二次确认。

### 模块三：磁盘占用深度排查与安全移除模块
核心目标：彻底解决「磁盘正在使用中，无法推出」的行业痛点，实现100%安全移除，保护磁盘硬件与数据安全。
1. **标准化一键安全移除全流程**
    严格遵循「数据落盘→占用释放→安全卸载→硬件弹出」的安全流程，绝对不跳过任何环节，杜绝强制断电风险：
    - 第一步：锁定磁盘写入权限，强制刷新所有写入缓存到磁盘，确保所有数据完全落盘；
    - 第二步：深度全量扫描占用该磁盘的所有进程，覆盖Finder窗口、预览程序、终端shell、正在编辑的文档、Spotlight索引、Time Machine备份、杀毒软件扫描、后台服务等全场景，无遗漏；
    - 第三步：分级智能释放占用，兼顾安全与便捷：
      - 一级无感释放：自动关闭该磁盘的Finder窗口、退出文件预览进程、终止终端目录访问、停止临时Spotlight索引，全程无弹窗、不影响用户其他操作；
      - 二级用户确认释放：针对用户正在使用的进程（如正在编辑的磁盘内文档、正在进行的文件传输），明确提示「XX进程（PID:xxx）正在使用XX文件，是否关闭该进程以完成安全移除？」，用户可选择关闭/跳过；
      - 三级系统进程谨慎处理：针对系统级进程（如Time Machine、fsck），仅做提示，不自动强制结束，避免系统异常，由用户手动确认是否停止相关任务。
    - 第四步：所有占用释放完成后，先执行macOS原生diskutil unmount安全卸载，再执行diskutil eject硬件弹出，确保磁盘完全断电，磁头归位；
    - 第五步：弹出成功后推送系统通知+提示音，明确告知用户「磁盘XX已安全移除，可放心拔出」。
2. **占用进程可视化管理**
    - GUI界面可展开查看占用磁盘的全量进程列表，展示进程名、PID、占用的文件路径、进程类型（用户/系统）、启动时间，支持手动结束单个进程，或一键结束所有非系统进程。
    - 「强制结束进程」功能默认隐藏于高级选项，附带强风险提示：「强制结束进程可能导致未保存的数据丢失，仅在无法正常移除时使用」，避免用户误操作。
3. **安全强制移除兜底方案**
    针对常规释放无法完成的极端场景，提供「安全强制移除」选项，区别于市面工具的直接断电：先锁定磁盘写入、刷新所有缓存到磁盘、以只读模式重新挂载确保无写入操作，再执行卸载与弹出，最大程度保证数据完整性与文件系统安全，杜绝磁盘损坏风险。
4. **附加增值功能**
    - 批量安全移除：支持同时选中多个外接磁盘，一键批量安全移除，适配多磁盘同时使用的专业用户；
    - 机械硬盘休眠保护：安全移除后，自动向机械硬盘发送休眠指令，让磁盘停止转动、磁头归位，减少硬件损耗，延长硬盘使用寿命；
    - 全局快捷键支持：可自定义一键批量移除所有磁盘、一键打开菜单栏面板的全局快捷键，适配键盘流用户。

### 模块四：GUI交互模块
核心目标：遵循macOS原生设计规范，操作路径最短，新手零门槛，老手不繁琐，体验流畅稳定，无冗余打扰。
整体设计原则：贴合macOS HIG设计规范，完美适配深色/浅色模式，无广告、无冗余弹窗、无无效元素，核心功能一键可达，兼顾轻量化与完整性。

#### 1. 菜单栏常驻入口（用户核心操作入口）
- 位置：macOS顶部菜单栏，图标简约轻量化，可自定义是否显示，默认开启；
- 下拉菜单核心布局：
  - 顶部：已插入的外接磁盘列表，每个磁盘展示卷标、可用空间、读写状态标签，右侧配备「打开」「安全移除」两个核心按钮，一键操作；
  - 中间：快捷功能区，包含「刷新磁盘列表」「批量安全移除所有磁盘」「挂载所有未挂载磁盘」；
  - 底部：「打开主界面」「设置」「退出应用」；
- 状态反馈：磁盘挂载成功/失败、移除成功/失败时，图标附带极简状态提示（如绿色小点=正常，红色感叹号=异常），不突兀但可清晰感知。

#### 2. 主应用窗口（完整功能入口）
采用macOS原生侧边栏+主内容区的经典布局，支持全屏、分屏、窗口大小自由调整，整体分为5个核心页面：
- **侧边栏导航**：固定展示「磁盘列表」「挂载规则」「操作日志」「磁盘健康」「设置」5个一级菜单，选中菜单下方展示对应快捷入口，切换流畅无卡顿。
- **① 磁盘列表详情页（默认首页）**
  顶部：搜索框（支持按卷标/UUID搜索磁盘）、刷新按钮、批量操作按钮；
  主体：卡片式布局，每个磁盘对应一张独立卡片，卡片内容层级清晰：
  - 顶部：磁盘卷标、文件系统格式、硬件接口、读写状态标签（可读写/只读/异常）；
  - 中间：环形容量进度条，直观展示总容量、已用/可用空间；
  - 底部：核心操作按钮「打开磁盘」「读写/只读模式切换」「安全移除」「查看占用进程」，高级选项折叠隐藏，展开后可查看磁盘详情、挂载到指定目录、执行文件系统检查、导出SMART信息；
  交互细节：卡片hover高亮，双击直接打开磁盘根目录，异常磁盘自动标红置顶。
- **② 挂载规则页**
  列表式展示用户已创建的所有挂载规则，每条规则展示目标磁盘、触发条件、执行动作，支持新建/编辑/删除/启用/禁用规则；
  新建规则采用向导式流程，新手零门槛：第一步选择目标磁盘→第二步设置触发条件（插入时/开机时/手动触发）→第三步设置挂载参数（读写模式、挂载路径、权限配置）→第四步设置附加动作（自动执行脚本、通知提醒）→第五步保存生效。
- **③ 操作日志页**
  全量记录所有磁盘操作日志，包含时间、操作类型、磁盘名称、操作结果、详细信息，支持按时间、磁盘、操作类型筛选，支持一键导出全部日志为TXT文件，方便问题排查；
  日志默认保存30天，用户可自定义保存时长，支持一键清空。
- **④ 磁盘健康页**
  针对支持SMART协议的磁盘，可视化展示完整健康数据：健康状态评级、通电次数、通电时长、坏道数量、磁盘温度、读写错误率等关键指标；
  异常指标自动标红告警，支持一键导出SMART健康报告，可设置温度阈值、健康异常自动告警。
- **⑤ 设置页**
  采用Tab分页布局，分类清晰，避免设置项混乱：
  - 通用设置：开机自启、菜单栏图标显示、外观模式、通知开关、提示音开关；
  - 挂载设置：默认挂载模式、挂载前自动文件系统检查开关、NTFS挂载参数配置、自动挂载黑名单管理；
  - 安全移除设置：自动释放非系统进程开关、移除后机械硬盘自动休眠、批量移除二次确认、强制移除风险提示开关；
  - 高级设置：日志保存时长、全局快捷键配置、扩展文件系统支持开关、调试模式开关；
  - 关于页面：软件版本、开源仓库地址、更新检查、贡献指南、用户反馈渠道。

#### 3. 弹窗与通知规范
- 所有系统通知均支持点击跳转对应操作，如挂载成功通知点击打开磁盘，移除失败通知点击打开占用进程详情页；
- 所有高危操作（强制结束进程、强制移除、格式化）均有二次确认弹窗，明确提示风险，默认选中「取消」，避免误操作；
- 无无关弹窗打扰，更新检查默认手动触发，不自动弹出更新提示。

### 模块五：高阶扩展与自定义模块
核心目标：满足专业用户、开源社区用户的高阶需求，提升产品扩展性与复用性。
1. **配置文件导入导出**：支持用户将挂载规则、全局设置导出为配置文件，可在多台Mac设备间一键导入同步，适配多设备用户与团队标准化部署场景。
2. **脚本自动化支持**：支持设置「挂载成功后」「移除前」的自定义Shell/Python脚本触发，实现自动备份、文件同步、服务启停等自动化操作，满足专业用户的定制化需求。
3. **配套命令行工具**：除GUI外，同步提供同名命令行工具，支持终端全功能操作，核心命令包括：
  ```bash
  # 挂载指定磁盘
  mountguard mount /dev/disk2 --mode rw
  # 安全移除指定磁盘
  mountguard eject /dev/disk2
  # 查看磁盘占用进程
  mountguard ps /dev/disk2
  # 列出所有外接磁盘
  mountguard list
  ```
  适配服务器、远程操作、自动化脚本场景，支持开源社区基于命令行工具二次开发。
4. **多语言支持**：默认支持简体中文、英文，预留多语言扩展接口，支持社区贡献其他语言翻译。

### 模块六：日志与运维保障模块
核心目标：实现问题可追溯、异常可预警，保障产品长期稳定运行，降低用户排查成本。
1. **全操作审计日志**：所有磁盘挂载、卸载、读写操作、配置修改、进程结束行为全量记录，不可篡改，出现问题可完整回溯。
2. **错误日志自动收集**：挂载失败、移除失败、系统报错时，自动收集相关上下文日志，用户可一键打包导出，提交至GitHub Issue，方便开发团队快速定位修复问题。
3. **异常告警机制**：磁盘健康异常、温度过高、文件系统错误、挂载失败时，自动推送系统通知告警，提前预警数据丢失风险。
4. **离线可用保障**：除用户手动点击「检查更新」外，无任何网络请求，全功能离线可用，绝对不收集、上传任何用户隐私数据与磁盘信息，符合开源透明与隐私合规要求。

## 五、技术实现核心原则（开发团队必须严格遵守）
1. **安全第一不可突破原则**
    - 所有磁盘操作100%基于macOS原生系统框架与官方命令，绝对不使用未公开的私有API，不修改系统文件，不注入系统进程，不安装强制内核扩展；
    - 所有写入操作必须先完成文件系统一致性检查，检查不通过禁止写入，强制只读挂载；
    - 所有安全移除操作必须先完成数据缓存落盘、文件系统卸载，再执行硬件弹出，绝对禁止直接磁盘断电；
    - 绝对不自动结束系统核心进程，所有系统级操作必须经过用户二次确认。
2. **轻量高性能原则**
    - 后台常驻进程空闲内存占用＜50MB，CPU占用＜1%，全事件驱动，无轮询操作，不影响Mac续航与系统性能；
    - 原生模式下，磁盘读写性能损耗＜5%，达到磁盘硬件本身的最大传输速度，无额外性能损耗。
3. **开源与兼容原则**
    - 代码完全开源，采用MIT开源协议，所有依赖均为开源组件，无闭源第三方库捆绑，支持用户自由编译、修改、分发；
    - 原生双架构支持，适配macOS 12.0及以上所有版本，提前预留新系统、新硬件的适配空间。
4. **隐私保护原则**
    - 零隐私数据收集，无后台网络请求，无用户行为统计，全功能离线可用，彻底杜绝隐私泄露风险。

## 六、用户核心使用流程
### 新手用户极简流程（零门槛）
1. 下载DMG安装包，双击打开，将MountGuard拖入应用程序文件夹，完成安装；
2. 打开应用，按照提示授予「磁盘管理权限」，完成初始化，全程无需其他配置；
3. 插入移动硬盘/U盘，软件自动识别并以可读写模式挂载，点击推送的通知即可打开磁盘，直接读写文件；
4. 使用完成后，点击顶部菜单栏图标，找到对应磁盘，点击「安全移除」，软件自动释放占用并完成弹出，收到通知后即可拔出磁盘。

### 高级用户自定义流程
1. 安装完成后，打开主界面，进入「挂载规则」页面，为常用磁盘创建专属挂载规则与自动化脚本；
2. 进入设置页面，配置默认挂载参数、安全移除选项、全局快捷键，完成个性化配置；
3. 插入磁盘，软件自动按预设规则挂载并执行对应动作，可在主界面查看磁盘详情、健康状态、占用进程；
4. 安全移除时，可查看占用进程详情，手动结束指定进程，或使用批量移除功能完成多磁盘操作；
5. 可使用配套命令行工具，编写自动化脚本，实现定制化的磁盘管理需求。

## 八、测试与质量保障标准
为确保产品稳定、不损害磁盘与数据，开发团队必须严格执行以下测试标准：
1. **兼容性测试**：覆盖macOS 12.0-最新正式版，Apple Silicon M1-M4全系列、Intel Mac 2016-2020款全机型，确保无兼容性问题。
2. **格式兼容性测试**：覆盖所有支持的文件系统，测试不同格式磁盘的挂载、读写、移除全流程，无失败、无数据损坏。
3. **稳定性压力测试**：连续72小时挂载磁盘，反复读写10GB以上大文件，反复执行挂载/移除操作，无崩溃、无内存泄漏、无CPU异常占用。
4. **安全移除专项测试**：模拟全场景占用情况（Finder、文件编辑、终端、系统索引、备份进程等），测试安全移除功能的释放成功率、弹出成功率，确保无无法移除的场景。
5. **异常场景容错测试**：模拟磁盘未正常推出、文件系统损坏、坏道、中途拔盘、断电等异常场景，测试软件的容错能力，确保不损坏磁盘、不丢失数据。
6. **性能测试**：对比原生挂载的读写速度，确保软件读写性能损耗＜5%，无明显速度下降。
7. **安全红线测试**：所有高危操作均有二次确认，无法操作系统内置磁盘，无修改系统文件的行为，无数据损坏风险。
