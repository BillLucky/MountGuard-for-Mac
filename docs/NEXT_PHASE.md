# Next Phase

This phase is intentionally closed.

MountGuard is already useful for:

- seeing external disks clearly
- checking what blocks an eject
- running a safe self-test
- ejecting more safely

The next phase starts later, not now.

## Candidate Work

- verified one-way sync
- resumable large file copy
- checksum validation after copy
- copy session logs and retry
- UUID-based rules and saved preferences

## Rule

Do not start these features until the current disk workflow is stable in daily use.
