# MountGuard v0.1.0

## English

First public MVP release.

Highlights:

- native macOS GUI + menu bar entry
- safer eject flow with busy-process scan
- disk self-test limited to `.mountguard-selftest`
- English-first GUI with Chinese toggle
- English and Chinese documentation
- first DMG package
